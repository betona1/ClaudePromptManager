원격 PC 연동

윈도우 등 다른 PC에서 Claude Code 쓰면 이 서버에 자동 기록됩니다.

원격 PC 연동 가이드

1

다운로드

📦 cpm-hooks.zip 서버 주소 자동 설정됨
2

압축 풀기

C:\ 에 풀면 C:\cpm\ 폴더가 생깁니다.

C:\cpm\
on_prompt_remote.py
on_stop_remote.py
settings.json
README.txt
3

hooks 설정

위치: %USERPROFILE%\.claude\settings.json

settings.json 없는 경우

C:\cpm\settings.json 을 %USERPROFILE%\.claude\ 에 그대로 복사

이미 settings.json 있는 경우

기존 파일 열고 "hooks" 부분만 추가:

"hooks": { "UserPromptSubmit": [ { "type": "command", "command": "python C:\\cpm\\on_prompt_remote.py" } ], "Stop": [ { "type": "command", "command": "python C:\\cpm\\on_stop_remote.py" } ] }
4

기존 기록 가져오기

이전 기록이 있으면 한 번만 실행:

python C:\cpm\import_history.py

중복은 자동 건너뜁니다.

Claude Code 재시작하면 자동 기록. 서버: {{ server_url }}
안 되면?
  • Python 설치 확인 — python --version
  • 서버 접속 확인 — {{ server_url }}
  • 경로 확인 — C:\cpm\on_prompt_remote.py

GitHub Personal Access Token 만들기

1

GitHub Settings 이동

GitHub 로그인 → 우측 상단 프로필 아이콘 → Settings

2

Developer settings

왼쪽 사이드바 맨 아래 Developer settings 클릭

3

토큰 생성

Personal access tokens → Tokens (classic) → Generate new token (classic)

Fine-grained token도 가능하지만 classic이 간편합니다.
4

설정

  • Note: CPM Sync (용도 메모)
  • Expiration: 원하는 기간 (또는 No expiration)
  • Scopes: repo 체크 (레포 읽기용)
5

토큰 복사

Generate token 클릭 → ghp_ 로 시작하는 토큰을 복사

이 페이지를 떠나면 토큰을 다시 볼 수 없습니다. 바로 위 Token 필드에 붙여넣기 하세요.

Telegram Bot 만들기 (완전 초보용)

Part A. 봇 만들기 (2분)
1

BotFather 열기

아래 링크를 폰에서 누르세요. Telegram 앱이 바로 열립니다.

BotFather 열기 (폰에서 클릭)
PC에서 보고 있다면? 폰 Telegram 앱 열고 → 상단 검색 → BotFather 입력 → 파란 체크표시 있는 계정 선택
확인: BotFather와 채팅방이 열렸나요? → 하단에 START 또는 메시지 입력창이 보이면 OK
2

/newbot 보내기

채팅창에 아래를 그대로 입력해서 보내세요:

/newbot
확인: BotFather가 "Alright, a new bot. How are we going to call it?" 라고 답했나요? → 다음으로
3

봇 이름 입력 (2번 대답해야 함!)

여기가 제일 헷갈리는 부분! BotFather가 두 번 물어봅니다. 하나는 "이름", 하나는 "ID" 입니다.

첫 번째 질문 — "이름을 뭘로 할래?" (표시되는 이름)

Alright, a new bot. How are we going to call it? Please choose a name for your bot.
내 CPM 봇

아무거나 OK. 한글도 됩니다. 나중에 바꿀 수 있어요.

두 번째 질문 — "username을 뭘로 할래?" (고유 ID)

Good. Now let's choose a username for your bot. It must end in `bot`.
my_cpm_alert_bot
my_cpm_bot → OK
TestCpmBot → OK
joacham_notify_bot → OK
my_cpm → bot으로 안 끝남
내봇_bot → 한글 안 됨
cpm_bot → 너무 짧아서 이미 누가 씀
"Sorry, this username is already taken" 나오면 → 다른 이름 시도. 앞에 자기 이름이나 숫자 붙이면 됩니다.
확인: "Done! Congratulations on your new bot." 메시지가 왔나요? → 다음으로
4

Bot Token 복사

위 축하 메시지 안에 토큰이 들어있습니다:

Done! Congratulations on your new bot. You can now add a description ...

Use this token to access the HTTP API:
7123456789:AAH_abcDEF1234ghijKLMN5678opqr

You can use this token ...
숫자:영문 형태의 긴 문자열을 꾹 눌러서 복사하세요!
콜론(:) 포함해서 전체를 복사해야 합니다.

복사 못 했으면? BotFather에게 /mybots → 봇 선택 → API Token 누르면 다시 볼 수 있습니다.
확인: 메모장에 붙여넣기 해보세요. 숫자:영문 형태면 OK. 여기까지가 Part A 끝!
Part B. Chat ID 알아내기 (1분)
5

Chat ID란?

봇이 메시지를 보낼 대상(나)의 고유번호입니다.

username(@myname)이 아니고, 순수 숫자입니다 (예: 123456789)

가장 쉬운 방법

아래 링크를 눌러 @userinfobot을 여세요:

userinfobot 열기 (폰에서 클릭)

열리면 /start 또는 아무 메시지를 보내세요:

/start
@joacham
Id: 5765213489
First: Joacham
Lang: ko

Id: 뒤의 숫자를 복사하세요. 그게 Chat ID입니다.

확인: 숫자만 복사했나요? (공백, @ 없이 숫자만!) → 다음으로
Part C. 가장 많이 빠지는 함정!
6

봇에게 먼저 /start 보내기

이걸 안 하면 Test가 절대 안 됩니다!
Telegram 봇은 사용자가 먼저 말을 걸어야만 그 사용자에게 메시지를 보낼 수 있습니다.

Telegram 검색에서 내가 방금 만든 봇 (예: @my_cpm_alert_bot)을 찾아서:

/start

이것만 보내면 됩니다. 봇이 아무 응답 안 해도 정상입니다.

완료! 이제 CPM에서 봇이 당신에게 메시지를 보낼 수 있습니다.
Part D. CPM에 등록 (30초)
7

토큰 + Chat ID 입력

이 페이지 위의 Telegram 봇 연동 → [+] 버튼 클릭:

  • Bot Token: 4단계에서 복사한 토큰 (예: 7123456789:AAH_abc...)
  • Chat ID: 5단계에서 복사한 숫자 (예: 5765213489)

[추가] 클릭 → 토큰 자동 검증 → 봇 이름이 자동으로 채워짐

[Test] 버튼 → Telegram에 메시지가 오면 성공!

성공! Telegram에 "CPM Test Message"가 왔으면 설정 완료!
1. BotFather 열기 2. userinfobot 열기
그래도 안 되면? 체크리스트
  • 6단계를 했나요? 내 봇에게 /start 를 반드시 보내야 합니다
  • Chat ID가 순수 숫자인지 확인 (@username이 아님!)
  • 토큰 복사할 때 앞뒤 공백이나 줄바꿈이 섞이지 않았는지 확인
  • 토큰에 콜론(:)이 포함되어 있는지 확인 (예: 123:ABC)
  • 그룹에 보내려면? 봇을 그룹에 멤버로 초대하고, Chat ID에 -로 시작하는 그룹 ID 사용
