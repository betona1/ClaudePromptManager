원격 PC 연동

윈도우 등 다른 PC에서 Claude Code 쓰면 이 서버에 자동 기록됩니다.

원격 PC 연동 가이드

1

다운로드

📦 cpm-hooks.zip 서버 주소 자동 설정됨
2

압축 풀기

C:\ 에 풀면 C:\cpm\ 폴더가 생깁니다.

C:\cpm\
on_prompt_remote.py
on_stop_remote.py
settings.json
README.txt
3

hooks 설정

위치: %USERPROFILE%\.claude\settings.json

settings.json 없는 경우

C:\cpm\settings.json 을 %USERPROFILE%\.claude\ 에 그대로 복사

이미 settings.json 있는 경우

기존 파일 열고 "hooks" 부분만 추가:

"hooks": { "UserPromptSubmit": [ { "type": "command", "command": "python C:\\cpm\\on_prompt_remote.py" } ], "Stop": [ { "type": "command", "command": "python C:\\cpm\\on_stop_remote.py" } ] }
4

기존 기록 가져오기

이전 기록이 있으면 한 번만 실행:

python C:\cpm\import_history.py

중복은 자동 건너뜁니다.

Claude Code 재시작하면 자동 기록. 서버: {{ server_url }}
안 되면?
  • Python 설치 확인 — python --version
  • 서버 접속 확인 — {{ server_url }}
  • 경로 확인 — C:\cpm\on_prompt_remote.py

GitHub Personal Access Token 만들기

1

GitHub Settings 이동

GitHub 로그인 → 우측 상단 프로필 아이콘 → Settings

2

Developer settings

왼쪽 사이드바 맨 아래 Developer settings 클릭

3

토큰 생성

Personal access tokens → Tokens (classic) → Generate new token (classic)

Fine-grained token도 가능하지만 classic이 간편합니다.
4

설정

  • Note: CPM Sync (용도 메모)
  • Expiration: 원하는 기간 (또는 No expiration)
  • Scopes: repo 체크 (레포 읽기용)
5

토큰 복사

Generate token 클릭 → ghp_ 로 시작하는 토큰을 복사

이 페이지를 떠나면 토큰을 다시 볼 수 없습니다. 바로 위 Token 필드에 붙여넣기 하세요.
